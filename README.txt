=== Brick Push 推箱子游戏 ===

🎮 游戏已启动！

✅ 游戏状态：
- ✅ Node.js 已更新到 v24.14.1
- ✅ 游戏服务器正在运行
- ✅ 游戏可以在浏览器中访问

🔗 访问方式：
1. 主游戏页面：http://localhost:3000
2. 游戏说明页面：http://localhost:8080/direct-run.html
3. 原始页面：http://localhost:3000/index.html

🎯 游戏操作：
- 移动：WASD 或 方向键
- 触摸控制：移动端按钮
- 目标：将方块推到指定位置

⚡ 游戏特性：
- 推各种类型的方块（普通、星星、心形、炸弹）
- 两种胜利条件：心心集合 或 敌人全灭
- 得分系统：破碎方块、压死敌人、收集物品
- 完整音效和视觉反馈

🛠️ 如何停止：
在终端中按 Ctrl+C 停止游戏服务器

📁 项目结构：
- src/ - 游戏源代码
- public/ - 游戏资源文件
- libs/ - 本地依赖包
- 各种配置文件

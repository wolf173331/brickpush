🚀 Supabase连接测试

为BrickPush游戏配置在线排行榜和多人联机

1 注册Supabase

访问 supabase.com 注册账号

推荐使用GitHub账号登录，创建免费项目

⏳ 等待注册

2 获取API密钥

登录Supabase后，创建新项目：

  1. 点击 "New Project"
  2. 填写项目名称: brickpush-game
  3. 设置数据库密码: 记住这个密码
  4. 选择区域: 建议选择亚洲区 (Singapore)
  5. 点击 "Create new project"

需要获取的密钥：

1. Project URL: https://[project-id].supabase.co

2. anon/public API key: eyJh... (以eyJ开头)

3 填写API密钥

将你的Supabase密钥填入下方进行测试：

// 测试结果将显示在这里

🔧 环境变量配置

测试成功后，在你的项目中创建 .env.local 文件：

VITE_SUPABASE_URL=你的Supabase_URL VITE_SUPABASE_ANON_KEY=你的anon_key

🎯 下一步操作

  1. 完成Supabase注册和项目创建
  2. 测试连接成功
  3. 创建数据库表结构
  4. 修改游戏代码集成Supabase
  5. 测试在线排行榜功能

⚠️ 安全提醒

  • 不要将API密钥提交到GitHub公开仓库
  • 使用 .env.local 文件存储密钥
  • 确保 .env.local 在 .gitignore 中
  • 仅使用anon key，不要使用service role key在客户端

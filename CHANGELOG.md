# Changelog

## [2026-04-03] 重大更新

### 新功能
- **关卡编辑器** (`level-editor.html`)：可视化编辑关卡地图，支持 File System Access API 直接保存到本地文件，无需手动复制
- **在线排行榜**：接入 Supabase，分数实时同步，失败时自动降级本地 localStorage
- **动态关卡加载**：游戏运行时通过 fetch 读取 `public/assets/levels.json`，修改关卡无需重新构建
- **最大关卡数控制**：`MAX_UNLOCKED_LEVELS` 常量控制开放关卡数，当前开放 15 关
- **启动脚本** (`start.sh`)：一键启动开发服务器并打开浏览器

### 游戏玩法改动
- **胜利条件**：仅保留心心连接通关，移除敌人全灭条件
- **时间限制**：随关卡递增难度，从第1关99秒线性递减到第20关60秒，警告阈值15秒
- **倒计时 UI**：移至顶部居中，警告时脉冲缩放，颜色三段变化（青→橙→红）

### 敌人系统重构
- **动态波次刷新**：敌人不再静态布置在地图，改为动态刷新，每关多波，波次配置写入关卡 JSON
- **四种敌人差异化 AI**：
  - FROG：随机游荡
  - BLOB：主动追玩家，可走进玩家格触发碰撞
  - BOW：追玩家，遇方块可跳越
  - GEAR：追玩家，遇星块/心心可推动（每次仅推一格）
- **碰撞后 stun**：敌人撞到玩家后随机游荡 1.2 秒
- **移动动画平滑**：改用 tween 位移插值，消除高速移动时的闪烁

### 分数系统重构（街机风格）
- 推坏普通方块 +100，推坏墙 +200，推坏星块 +300
- 杀怪基础 +500，5 秒内 combo 每次递增 +500，上限 10000
- 通关奖励 = 关卡编号（第1关+1，第15关+15），从末尾数字可读出通关关数
- 0 分不允许提交排行榜

### 玩家受伤反馈
- 受伤后快速闪烁 3 次（70ms/次），视觉冲击更明显
- 受伤后 300ms 锁定输入，防止连续受伤

### 炸弹机制
- 玩家推炸弹时，路径上遇到敌人立即在敌人位置爆炸，炸弹不穿过敌人

### 素材更新
- `block.svg`：重新设计为四方无缝拼接，相邻方块视觉融合成墙
- `grass.svg`：重新设计为俯视草地砖块风格，有草叶纹理
- `bomb.png`：新增炸弹 PNG 素材

### 关卡数据
- 全部 20 关重新设计，按有机走廊风格布局，方块密度随关卡递增
- 关卡 JSON 新增 `enemyWaves`、`enemiesPerWave` 字段


### 工程清理
- 删除所有无关文件（旧部署脚本、测试 HTML、CloudStudio 文档等 29 个文件）
- 新增 `README.md` 统一说明文档
- `tsconfig.json` 加入 `vite/client` 类型，修复 `import.meta.env` 类型错误

📋 GitHub Pages配置步骤

  1. 访问: https://github.com/wolf173331/brickpush
  2. 点击仓库右上角的 Settings
  3. 左侧菜单选择 Pages
  4. 在 Build and deployment 部分:
    • Source: 选择 Deploy from a branch
    • Branch: 选择 main
    • Folder: 选择 /docs
  5. 点击 Save
  6. 等待1-2分钟构建完成

## 🎯 目标
让你的朋友可以玩到你的BrickPush推箱子游戏！

## 📱 三种分享方式

### 方式1：GitHub Pages（推荐）
**优点**：免费、永久、简单
**步骤**：
1. 按照《GitHub Pages部署指南》部署游戏
2. 获得你的游戏URL：`https://你的用户名.github.io/brickpush`
3. 分享这个链接给朋友

**分享内容**：
```
🎮 来玩我做的推箱子游戏！

游戏名：BrickPush
链接：https://你的用户名.github.io/brickpush

🌟 游戏特点：
✅ 3个不同关卡
✅ 心心方块必须连接才能过关
✅ 3点生命值系统
✅ 99秒倒计时
✅ 排行榜系统

📱 支持手机和电脑游玩
```

### 方式2：本地临时分享
**优点**：快速、无需部署
**步骤**：
1. 在本地启动服务器：
   ```bash
   cd "/Users/ludi/workspace/new work/brickpush"
   npm run start:server
   ```

2. 获取你的公网IP：
   - 方法1：在浏览器访问 https://whatismyipaddress.com
   - 方法2：在终端输入 `curl ifconfig.me`

3. 配置路由器端口转发（如果需要）：
   - 将端口8080转发到你的电脑

4. 分享链接：`http://你的公网IP:8080`

**注意**：
- 你的电脑需要一直开机
- 你的网络需要有公网IP
- 建议临时分享使用

### 方式3：使用在线代码编辑器
**优点**：无需服务器配置
**选项**：
1. **CodePen**：https://codepen.io
2. **JSFiddle**：https://jsfiddle.net
3. **Replit**：https://replit.com

## 📝 常见问题解答

### Q1：游戏打不开怎么办？
**A：**
- 检查URL是否正确
- 清理浏览器缓存
- 换一个浏览器试试

### Q2：游戏卡顿怎么办？
**A：**
- 关闭其他占用资源的程序
- 清理浏览器缓存
- 使用Chrome浏览器

### Q3：游戏进度会保存吗？
**A：**
- 排行榜数据会保存
- 关卡进度不会自动保存
- 建议截图记录高分

### Q4：如何报告bug？
**A：**
- 截图保存错误信息
- 记录出现问题的步骤
- 告诉我你使用的设备
